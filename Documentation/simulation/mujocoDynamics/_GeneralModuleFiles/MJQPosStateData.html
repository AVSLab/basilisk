Module: MJQPosStateData

class MJQPosStateData : public StateData
#include <MJQPosStateData.h>

Class representing the entire joint state data in a MuJoCo simulation.

This class manages the position states (qpos) and their derivatives in a MuJoCo simulation. It extends StateData to include specific configurations and propagation logic for position states.
