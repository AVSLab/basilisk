MJGeomInfo

struct MJGeomInfo
#include <MJGeomInfo.h>

Describes the geometry (shape, size, position, orientation, color) of a single MuJoCo geom, suitable for visualization.

The type field stores a mjtGeom integer value from mujoco/mujoco.h. Common values: mjGEOM_SPHERE=2, mjGEOM_CAPSULE=3, mjGEOM_ELLIPSOID=4, mjGEOM_CYLINDER=5, mjGEOM_BOX=6.
