Module: THRTimePair

struct THRTimePair
#include <THRTimePair.h>

Container for time/value pairs for ramp on and ramp off profiles.

This structure is used to build up the thruster on and thruster off ramps. It pairs the time-delta from on command and thrust factor (percentage/100.0) for the entire ramp.
