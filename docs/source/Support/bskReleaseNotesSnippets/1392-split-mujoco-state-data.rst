- Refactored MuJoCo state ownership. Each ``MJScalarJoint``, ``MJBallJoint``, and ``MJFreeJoint`` now registers its own qpos/qvel ``StateData`` (translation and attitude split for free joints) so the adaptive integrator computes a separate error norm per joint at its natural scale. This fixes cases where orbital translation magnitudes dominated the unified state norm and let stiff hinge dynamics drift unflagged.
- Added ``QuaternionStateData`` in ``simulation/dynamics/_GeneralModuleFiles`` for unit-quaternion attitude DOFs with exponential-map propagation and renormalization. This replaces the removed ``MJQPosStateData`` / ``MJQPosAttitudeStateData`` wrappers.
