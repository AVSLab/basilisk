- Importing :ref:`unitTestSupport` (directly or indirectly through utilities such as ``simIncludeGravBody`` and ``vizSupport``) no longer requires ``pytest`` at runtime. ``pytest`` is now an optional import; the two ``pytest.approx``-based helpers ``compareList`` and ``compareListRelative`` raise a clear error only if they are actually called without ``pytest`` installed. This allows scripts that use the shipped Basilisk wheels to import these utilities without installing the test-only ``pytest`` dependency.
