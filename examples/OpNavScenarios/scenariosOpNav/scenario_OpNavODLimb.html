7. scenario_OpNavODLimb

7.1. Overview

This scenario only performs the orbit determination component of the FSW stack. It uses the Limb based method for image processing. More details can be found in Chapter 4 of Thibaud Teil’s PhD thesis.

The script can be run at full length by calling:

python3 scenario_OpNavODLimb.py
